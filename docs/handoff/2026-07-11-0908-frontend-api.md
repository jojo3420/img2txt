### 사용자 원본 요청
> "claude/ollama 백엔드 실측 배치 검증 → 프런트 연동(실제 API) → E2E 진행해"

### 사용자 제약-금지사항 (반드시 준수)
🔴 반드시:
- 길이 가드는 corrector 오케스트레이션 책임, 백엔드는 순수 보정만 (뒤집지 말 것)
- 센티넬 프로토콜(`===문단 N===`) 유지, 개수 요약 마커는 폐기됨(부활 금지)
- Silent Failure 금지: 보정 실패(예외/개수 불일치)는 KEPT가 아닌 FAILED로 기록
- 결정 사항 표(아래)는 뒤집지 않음 — 변경 시 사용자 확인
- 기존 `img2txt` 순수 로직(scanner/ocr/layout/assembler/corrector/writer)은 거의 수정 없이 재사용 (스펙 1절)

⚠️ 환경 주의:
- OCR은 Apple Vision(`ocrmac`)이라 macOS 전용. `ocrmac` 미설치 시 OCR 통합 테스트(`test_recognize_real_page`) 실패 — 보정 경로와 무관한 사전 존재 실패
- pytest/mypy는 `rtk` 프록시 경유로 실행해야 함 (`rtk pytest ...`). 직접 `python3 -m pytest`는 pytest 미설치로 실패. 프로젝트 `.venv`(Python 3.13) 사용
